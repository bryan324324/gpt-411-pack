# GPT 411 Pack

A lightweight static web app that generates clean ChatGPT commands for property analysis.

## What it does

- Generates a one-line structured command for ChatGPT
- Supports Full Report or Comps Only
- Supports Residential, Multifamily 5+, Commercial, and Land
- Optional notes field
- Works on Mac, PC, and phone

## Suggested workflow

1. Enter the address
2. Choose property type
3. Choose mode
4. Click Generate Command
5. Click Copy
6. Open ChatGPT
7. Paste and send
